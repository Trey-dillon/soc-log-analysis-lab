# 📌 Overview

This project documents my hands-on practice analyzing security logs and investigating events from a Security Operations Center (SOC) perspective.

The goal of this lab is to develop skills in security monitoring, threat detection, and incident investigation.

# Investigation Scenarios

## Failed Login Investigation

Scenario:

Multiple failed authentication attempts detected.

Analysis:

- Reviewed login events
- Identified affected accounts
- Determined potential causes

---

## Unauthorized Access Review

Scenario:

Suspicious account activity identified.

Investigation steps:

1. Review event logs
2. Identify user activity
3. Analyze timestamps
4. Document findings
