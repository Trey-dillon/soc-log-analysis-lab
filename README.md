# 📌 Overview

This project documents my hands-on practice analyzing security logs and investigating events from a Security Operations Center (SOC) perspective.

The goal of this lab is to develop skills in security monitoring, threat detection, and incident investigation.

# Screenshots

## 1. Event Viewer Overview

![Event Viewer Overview](screenshots/SOC%201%20Event%20Viewer%20Overview.png)

Opened Windows Event Viewer and navigated to the **Windows Security** log to review authentication events and other security-related activity generated by the operating system.

---

## 2. Successful Login Event (Event ID 4624)

![Successful Login Event](screenshots/SOC%202%20Successful%20Login%20Event.png)

Investigated **Event ID 4624**, which records successful user authentication. Reviewed details such as the account name, logon type, and timestamp to understand how Windows records successful logins.

---

## 3. Failed Login Event (Event ID 4625)

![Failed Login Event](screenshots/SOC%203%20Failed%20Login%20Event.png)

Analyzed **Event ID 4625** after intentionally generating failed login attempts. Examined the failure reason, affected account, and event details to understand how unsuccessful authentication attempts are logged.

---

## 4. Filtering Security Logs

![Filtering Security Logs](screenshots/SOC%204%20Filter%20Security%20Logs.png)

Applied Event Viewer filters to display only selected Security Event IDs, allowing faster investigation by reducing unrelated log entries.

---

## 5. PowerShell Event Log Query

![PowerShell Event Log Query](screenshots/SOC%205%20PowerShell%20Log%20Query.png)

Used PowerShell to query Windows Security Event Logs from the command line, demonstrating an alternative method for investigating authentication activity and reviewing recent security events.
